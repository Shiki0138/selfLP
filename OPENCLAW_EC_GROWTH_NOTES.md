- AIエージェント（OpenClaw + Claude Code + antigravity）を活用し、運用を半自動化する。

## 基本戦略（3フェーズ）

### Phase 1（0〜30日）: 受け皿整備
1. Instagram運用（営業色ゼロ）
- 保存・シェアされる教育投稿を週4本

- 悩み別導線

- FAQ

- 月8本目標（悩み特化記事）

3. 自動化強化
- 投稿草案、週次レポート、改善提案をAIで半自動化

## KPI/KGI設計

### KGI（6ヶ月目安）
- EC月商: 20万円 → 100万円
- 卸売上: 10万円 → 30万円

### 重点KPI
- ECセッション数
- CVR
- AOV（平均注文単価）
- 初回→2回目購入率（最重要）
- LINE登録率
- 広告ROAS

## クリエイティブ方針
- 投稿は「売る」より「納得を積む」
- 保存価値の高い実用コンテンツ中心
- セールス投稿は避け、教育・共感・信頼形成を優先

## AI運用体制（役割分担）
- OpenClaw: 司令塔（要件整理、タスク分解、定期レポート通知）
- Claude Code: 実装担当（EC/LP/LINE/計測）
- antigravity: 実験担当（新施策PoC、クリエイティブ比較）

## 今週の最小バックログ
1. LINEステップ（追加後3通 + 購入後3通）
2. 商品ページ改善（悩み別導線・FAQ・セット導線）
3. 計測実装（UTM・CV・再購入イベント）
4. Instagram投稿テンプレ7本作成
5. 週次KPIレポートの自動投稿化（Discord）

## 判断ルール
- 最初の2週間は完全自動投稿しない（承認制）
- KPI未達なら自動化範囲を広げない
- 投稿数ではなく「保存率・シェア率・2回目購入率」を評価軸にする
