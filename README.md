# プレミアムボタニカルシャンプー LP

シャンプー商品向けのランディングページ（LP）プロジェクトです。
画像重視×スクロールアニメーション×高級感をコンセプトに設計されています。

## 📋 プロジェクト概要

- **商品**: プレミアムボタニカルシャンプー
- **ターゲット**: 30〜50代女性、髪のダメージ・悩みを抱える層
- **目的**: EC購入誘導（定期便契約）
- **構成**: 汎用テンプレート11セクション準拠

## 🎯 主な特徴

- ✅ 11セクション構成（ファーストビュー〜フッターCTA）
- ✅ GSAP + ScrollTriggerによる高度なアニメーション
- ✅ Lenisによるスムーススクロール
- ✅ Before/After比較スライダー
- ✅ FAQ アコーディオン
- ✅ レスポンシブ対応
- ✅ プレースホルダーベースの設計（画像差し替え可能）

## 🚀 使い方

### 1. ローカルサーバーで起動

```bash
# シンプルなHTTPサーバー
python -m http.server 8000

# または Node.js の場合
npx serve
```

ブラウザで `http://localhost:8000` を開く

### 2. ファイル編集

- **HTML**: `index.html` を編集
- **スタイル**: `css/` フォルダ内のCSSファイルを編集
- **アニメーション**: `js/main.js` を編集

### 3. 画像の差し替え

現在はプレースホルダー（色付きボックス）で表示されています。
実際の画像に差し替えるには：

1. `IMAGE_GENERATION_PROMPTS.md` を参照
2. AI画像生成ツール（Midjourney/DALL-E等）でプロンプト実行
3. 生成した画像を `images/placeholders/` に配置
4. HTMLの `*-placeholder` クラスを `<img>` タグに置き換え

`js/main.js` の各GSAPアニメーション設定を変更
```javascript
gsap.from(element, {
  duration: 1,    // アニメーション時間
  delay: 0.5,     // 遅延
  ease: 'power2.out', // イージング
});
```
