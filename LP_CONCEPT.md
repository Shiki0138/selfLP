# シャンプーLP制作構想書

## プロジェクト概要
**商品**: プレミアムシャンプー
**LP種別**: 商品販売型（EC誘導）
**コンセプト**: 画像重視×スクロールアニメーション×高級感

---

## 技術スタック提案

### フロントエンド基盤
- **HTML5**: セマンティックなマークアップ
- **CSS3**: Grid/Flexbox、カスタムプロパティ活用
- **Vanilla JavaScript**: 軽量化重視

### アニメーションライブラリ
1. **GSAP (GreenSock) + ScrollTrigger**
   - 業界標準、パフォーマンス最高クラス
   - 複雑なタイムライン制御が可能
   - 推奨度: ★★★★★

2. **Lenis (スムーススクロール)**
   - 高級感のある滑らかなスクロール体験
   - GSAPとの相性抜群
   - 推奨度: ★★★★★

3. **AOS (Animate On Scroll)**
   - シンプルな実装、軽量
   - 基本的なフェードイン・スライドインに最適
   - 推奨度: ★★★★☆

### 画像戦略
- **WebP形式**: 高画質×軽量化
- **遅延読み込み**: Intersection Observer API
- **レスポンシブ画像**: srcset/sizes属性
- **視差効果**: パララックススクロール

## スクロールアニメーション設計

### 全体フロー
```
[ヒーローエリア]
↓ スクロール開始
[悩み提示] ← フェードイン + スライド
↓
[解決策] ← パララックス背景
↓
[Before/After] ← 画像比較スライダー
↓
[3つの特徴] ← 3D回転カード
↓
[使用方法] ← タイムライン展開
↓
[レビュー] ← カルーセル
↓
[成分詳細] ← フリップカード
↓
[FAQ] ← アコーディオン
↓
[価格] ← プランカード比較
↓
[CTA] ← パルスボタン
```

### アニメーショントリガー
- **開始位置**: 要素が画面の80%位置に到達
- **終了位置**: 要素が画面の20%位置を超える
- **イージング**: ease-out（自然な減速）
- **所要時間**: 0.6〜1.2秒

## パフォーマンス最適化

### 画像最適化
- WebP形式 + JPEGフォールバック
- 遅延読み込み（loading="lazy"）
- 適切なサイズ（2x Retina対応）

### JavaScript最適化
- モジュール分割
- 非同期読み込み
- Intersection Observer でビューポート監視

### CSS最適化
- CSSカスタムプロパティで管理
- will-change プロパティで事前最適化
- GPU加速（transform, opacity優先）
