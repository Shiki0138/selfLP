## 3レイヤーファネル構造

### 第1層：認知（TOFU）── 動画視聴・保存

| 項目 | 設定 |
|------|------|
| **目的** | 動画視聴・コンテンツ保存 |
| **ターゲット** | ブロード配信（25-45歳女性）+ 美容関心層 |
| **予算配分** | 全体の **30%** |
| **KPI** | 動画視聴単価 ¥3以下、保存率 |
| **最適化イベント** | ThruPlay（動画15秒以上視聴） |

**クリエイティブ方針：**
- 「市販シャンプーで髪が傷むNG習慣」等の**問題提起リール**
- 「美容師が教える正しいシャンプーの仕方」等の**教育コンテンツ**
- オーガニックInstagramリールで高保存率の投稿を広告へ流用

**週次アクション：**
1. 毎週月曜 → オーガニック投稿の保存率トップ3を特定
2. 水曜までに → 広告用にリサイズ
3. 木曜に → 第1層キャンペーンへ投入

---

### 第2層：検討（MOFU）── LP訪問

| 項目 | 設定 |
|------|------|
| **目的** | LP訪問（`/lp/lp01.html`） |
| **ターゲット** | 第1層動画視聴者の類似1-3%、IGエンゲージメント層 |
| **予算配分** | 全体の **30%** |
| **KPI** | CPC ¥80以下、LP滞在時間 |
| **最適化イベント** | Landing Page View |

**クリエイティブ方針：**
- 美容師による専門解説動画
- Before/After動画（サロン素材がベスト）
- 「サロン専売品質をご自宅で」の訴求

**リンク先設定：**
```
https://self138.com/lp/lp01.html?utm_source=instagram&utm_medium=paid&utm_campaign=consideration&utm_content=[creative_name]
```

### 第3層：購入（BOFU）── CV（リターゲティング）

| 項目 | 設定 |
|------|------|
| **目的** | CV（購入完了） |
| **ターゲット** | LP訪問者、カート放棄者、過去購入者 |
| **予算配分** | 全体の **40%** |
| **KPI** | **CPA ¥3,000以下** |
| **最適化イベント** | Purchase |

**クリエイティブ方針：**
- SNSリアル口コミのスクリーンショット
- 期間限定オファー（あれば）
- サロン採用実績の念押し（「全国100店舗以上が選んだ理由」）
- 「リピート率91%」の数字を前面に

## ターゲティング最適化

### コアターゲット（以下に絞り込む）
- 「髪質改善」「縮毛矯正」「ダメージ補修」に関心を持つ層
- 美容院の予約履歴がある層

### 広告セット統合
- **広告セットを1つに統合** → 予算集中で学習高速化
- 複数の広告セットに分散させない

### 除外設定（必須）
- 「安い」「市販」「ドラッグストア」等の価格重視キーワード
- 競合ブランド名での検索者（無関係なクリック防止）

## Meta Ads Manager での設定手順

### Step 1: 既存キャンペーンの停止
1. Ads Managerにログイン
2. 既存の全キャンペーンを「オフ」に設定
3. 過去データは参照用に保持（削除しない）

### Step 2: 新キャンペーン作成
```
キャンペーン1: [self] 認知_動画視聴
├── 目的: 動画の再生数アップ
├── 予算: 日予算 ¥8,300（月25万円）
├── 広告セット: 1つのみ
│   ├── ターゲット: 25-45歳女性・美容関心
│   └── 配置: Instagram リール + フィード + ストーリーズ
└── 広告: 3〜5本の動画クリエイティブ

キャンペーン2: [self] 検討_LP訪問
├── 目的: トラフィック（ランディングページビュー）
├── 予算: 日予算 ¥8,300（月25万円）
├── 広告セット: 1つのみ
│   ├── ターゲット: 動画視聴者カスタムオーディエンス + 類似1-3%
│   └── 配置: Instagram フィード + ストーリーズ
└── 広告: Before/After + 専門解説

キャンペーン3: [self] 購入_リターゲティング
├── 目的: コンバージョン（購入）
├── 予算: 日予算 ¥11,000（月33万円）
├── 広告セット: 1つのみ
│   ├── ターゲット: LP訪問者 + カート放棄者
│   └── 配置: Instagram + Facebook 全配置
└── 広告: 口コミ + 限定オファー + 実績訴求
```

### Step 3: ピクセル・イベント確認
- Meta Pixel が `self138.com` と `shop.self138.com` の両方で発火していることを確認
- `Purchase` イベントが正しく計測されていることをEvents Managerで確認
- LP01の `add_to_cart` イベント（GTM経由）がMeta Pixelに連携されているか確認
