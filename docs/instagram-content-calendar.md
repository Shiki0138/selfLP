## リール → 広告の循環フロー

```
1. オーガニックリール投稿（週3本）
   ↓
2. 毎週月曜：保存率トップ3を特定
   ↓
3. 水曜まで：広告用にリサイズ・微調整
   ↓
4. 木曜：Meta広告 第1層（認知）へ投入
   ↓
5. 翌週：パフォーマンス計測
   → 高パフォーマンス → 第2層（検討）へも展開
   → 低パフォーマンス → 停止・新素材に差し替え
```

**効果：クリエイティブ制作費50%削減 + 検証済み素材で広告効率向上**
