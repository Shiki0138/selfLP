# @cosme・LIPS 運用計画

年商1億円マーケティング戦略 第10章に基づく

---

## 目的

@cosme と LIPS の口コミを戦略的に管理し、
**検索経由の自然CV** を増加させる。

**目標：口コミ100件以上**

---

## @cosme 運用

### 公式アカウント開設
1. [@cosmeブランドページ](https://www.cosme.net/) からブランド登録申請
2. 商品情報の正確な登録（成分表示、価格、販売チャネル）
3. 商品画像のアップロード（Amazon A+と同等の画像）

### 口コミ管理
| アクション | 頻度 | 内容 |
|----------|------|------|
| 口コミチェック | 毎日 | 新着口コミの確認 |
| 返信 | 48時間以内 | 丁寧な感謝 + 追加アドバイス |
| ネガティブ対応 | 即日 | 真摯な謝罪 + 改善策提示 |
| 商品情報更新 | 月1回 | 最新情報の反映 |

### 返信テンプレート

**ポジティブ口コミへの返信：**
```
口コミのご投稿ありがとうございます！
○○様のお悩みが改善されたとのこと、
開発者の東も大変喜んでおります。

selfの美容液泡パック、ぜひ続けてみてくださいね。
季節の変わり目のケアについてもお気軽にご質問ください✨
```

**ネガティブ口コミへの返信：**
```
貴重なご意見をいただきありがとうございます。
○○の点でご期待に添えず申し訳ございません。

selfは髪質やダメージの状態によって
最適な使い方が異なります。
もしよろしければ、LINE(@self138)にて
個別にアドバイスさせていただければと思います。

今後の商品改善に活かしてまいります。
```

---

## LIPS 運用

### アカウント開設
1. [LIPS](https://lipscosme.com/) でブランド登録
2. 商品登録（ヘアケアカテゴリ）
3. 商品画像・説明文の設定

### 運用方針
- @cosme と同様の口コミ管理
- LIPSは若年層（20-30代）の利用が多いため、カジュアルなトーンで返信

---

## 口コミ獲得施策

| 施策 | 方法 | 目標 |
|------|------|------|
| LINE購入者へのレビュー依頼 | ステップ配信で@cosme/LIPSへの投稿を依頼 | 月+10件 |
| 商品同梱QRコード | @cosme口コミページへのQRカード同梱 | 月+5件 |
| キスマイレビュー | @cosme公式の口コミ促進プログラム活用 | 月+5件 |
| SNS連携 | Instagram投稿から@cosme口コミへ誘導 | 月+3件 |
