# Amazon最適化ガイド

年商1億円マーケティング戦略 第4章に基づく

---

## Amazonの戦略的位置づけ

Amazonは「もう一つの売り場」ではなく、ブランド全体の **「信頼ブースター」** として再定義する。

| 役割 | 説明 |
|------|------|
| 新規獲得 | 低ハードル（既存アカウントで即購入）での初回接点 |
| レビュー資産 | 信頼の蓄積 → LP・広告の社会的証明に転用 |
| プラットフォームSEO | Amazon内検索での露出強化 |

**年間売上目標：2,500万円**（全体の25%）/ 月間約150件の新規

## 2. 箇条書き（Bullet Points）最適化

各商品の5つの箇条書きを以下の順序で最適化：

1. **ターゲット明示**
   「縮毛矯正・髪質改善をしている方、カラーやパーマでダメージが気になる方のために開発しました」

2. **独自成分の訴求**
   「約1分で髪内部に浸透するペリセアと、ドライヤーの熱で補修するエルカラクトンのW処方」

3. **使い方の差別化**
   「泡立てた後1〜2分置く"美容液泡パック"でサロン帰りの仕上がりに」

4. **開発背景**
   「髪質改善特化の美容師が、肌の弱い家族のために1年以上かけて処方設計」

5. **実績**
   「全国100店舗以上の美容室で導入。リピート率91%の実力」

## 3. A+コンテンツ（商品ページ画像）最適化

画像7枚の推奨順序：

| 枚目 | 内容 | 目的 |
|------|------|------|
| 1枚目 | メイン商品写真（白背景） | 第一印象 |
| 2枚目 | **対象者明示**（30代・産後・縮毛矯正経験者のイラスト/写真） | 「自分ごと」化 |
| 3枚目 | **Before/After** 写真 | 効果の可視化 |
| 4枚目 | **美容師の権威性**（東えりか氏の写真 + 100店舗導入実績） | 信頼性 |
| 5枚目 | **エルカラクトンの熱補修メカニズム** 図解 | 科学的根拠 |
| 6枚目 | **美容液泡パック** の使い方ステップ（1〜2分置く） | 特別感 |
| 7枚目 | **香りの訴求**（ベルガモット由来リラックスフローラル） | 感性訴求 |

## 5. レビュー獲得戦略

### 現在のレビュー数：31件（閾値の境界線上）

**目標：100件突破**（CVRが飛躍的に向上する閾値：50件→100件）

### 施策

| 施策 | 方法 | 目標 |
|------|------|------|
| Amazon Vine | 新バリエーション追加時にVineプログラム活用 | 質の高いレビュー20件 |
| フォローアップメール | 購入後7日でレビュー依頼メール | 返信率10%目標 |
| 商品同梱カード | パッケージにQRコード付きレビュー依頼カード | 月+5件 |
| SP広告集中投下 | レビュー付き購入を最大化 | ACOS 25%以下維持 |
