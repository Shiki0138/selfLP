# 美容師アンバサダー制度設計

年商1億円マーケティング戦略 Phase 3 に基づく

---

## 目的

self導入サロンの美容師10〜20名を **マイクロインフルエンサー** として起用し、
月間リーチ10万〜50万impを達成する。

---

## アンバサダーの定義

| 項目 | 内容 |
|------|------|
| 人数 | 10〜20名（段階的に拡大） |
| 資格 | self導入サロンの美容師 |
| フォロワー目安 | 1,000〜10,000人（マイクロ） |
| 活動内容 | 月2投稿以上（Instagram）|

---

## 選定基準

| 基準 | 重要度 | 評価方法 |
|------|-------|---------|
| selfへの理解・愛着 | ★★★ | インタビュー |
| Instagramのフォロワー数 | ★★ | アカウント確認 |
| 投稿の質（写真/動画） | ★★★ | 過去投稿レビュー |
| エンゲージメント率 | ★★★ | いいね率3%以上 |
| 髪質改善の施術頻度 | ★★ | サロンからの情報 |
| コミュニケーション力 | ★★ | インタビュー |

---

## 報酬設計

### ベーシック報酬

| 活動 | 報酬 |
|------|------|
| 月2投稿（Instagram） | self商品1点プレゼント（月1回） |
| 月4投稿以上 | self商品2点プレゼント + 卸価格10%OFF |
| アフィリエイトリンク経由のCV | 売上の10%（インセンティブ） |

### ボーナス報酬

| 条件 | 報酬 |
|------|------|
| 投稿がバズった（リーチ1万以上） | 現金 ¥10,000 or 商品セット |
| 素材が広告に採用された | 現金 ¥20,000 |
| 月間MVP | selgブランドとのコラボ商品開発への参加権 |

---

## アンバサダー向けガイドライン

### 投稿ルール
- **ハッシュタグ**：`#selfシャンプー #selfアンバサダー #PR` を必ず含める
- **メンション**：`@self_shampoo`（公式アカウント）をメンション
- **NG事項**：
  - 他ブランドの比較批判
  - 「○○フリー」等の虚偽表記（景品表示法違反）
  - 定価以外の価格の暗示
- **クリエイティブ自由度**：自身のスタイルで自由に投稿OK

### 投稿テーマ例
- Before/After（自サロンのお客様 ※同意取得前提）
- 「私がselfを導入した理由」
- 美容液泡パックのデモ動画
- お客様の感想紹介

---

## 運用フロー

```
月初：テーマ提案（任意）をアンバサダーに共有
  ↓
投稿：各自のペースで月2本以上
  ↓
月末：投稿実績の集計
  ↓
報酬：翌月初に商品発送 or インセンティブ支払い
  ↓
月次：MVPの選定・表彰
```
