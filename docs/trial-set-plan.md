# トライアルセット企画書

年商1億円マーケティング戦略 Phase 2 に基づく

---

## 目的

新規顧客の **購入ハードル** を劇的に下げ、CVRを2倍に向上させる。

現在の問題：
- セット ¥8,460 は初回購入には高い壁
- Amazon単品 ¥3,980 でも「失敗したくない」心理が働く
- → **¥2,980のミニセット** で「まず試す」ハードルを最小化

## LP01への組み込み案

現在の3択CTAに加えて、トライアルを追加：

```
A. 本気で髪質改善 → セット ¥8,460（最おすすめ）
```
