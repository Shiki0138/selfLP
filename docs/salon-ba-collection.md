# サロン Before/After 素材収集マニュアル

年商1億円マーケティング戦略 Phase 2 に基づく

---

## 目的

導入サロン10店舗から月10本の Before/After 動画・写真素材を安定的に収集し、
**広告・LP・SNSの全チャネル** で活用できる素材ライブラリを構築する。

---

## 収集する素材の種類

| 種類 | 形式 | 用途 |
|------|------|------|
| Before/After 写真 | 正面 + 横顔（自然光推奨） | LP01、Instagram、Amazon A+ |
| Before/After 動画 | 15-30秒（縦型） | Instagram リール、Meta広告 |
| お客様の声動画 | 30秒〜1分（顔出しor声のみ） | LP01、SNS、広告第3層 |
| サロンでの施術風景 | 15秒程度 | SNSブランディング |

---

## 撮影ガイドライン

### 写真

| 項目 | 基準 |
|------|------|
| 解像度 | 1080px以上（スマホOK） |
| 背景 | 白 or サロン内（シンプルな背景） |
| ライティング | 自然光がベスト（窓際推奨） |
| アングル | 正面 + 横（左右どちらか）の2パターン |
| Before | 施術前（乾いた状態）|
| After | 施術後（乾かし仕上がり）|
| 注意 | 同じ角度・距離・照明で統一 |

### 動画

| 項目 | 基準 |
|------|------|
| 解像度 | 1080×1920（縦型・9:16） |
| 長さ | 15〜30秒 |
| BGM | 不要（後から追加） |
| 構図 | Before → 施術中 → After の流れ |
| ナレーション | 不要（テロップで対応） |
